#Koa 2 API Boilerplate

### Development tools
* Visual Studio Code
* Web Storm

### Requirements
```
Node > v6.0
Typescript > 2.0.0.
Mongodb
```

### Build
1. Open project with editor (we prefer: Visual Studio Code)
```
npm install 
```
2. Start compiling
```
ts --watch
```

### Commands
Build application (Typescript to Javascript)
```
npm run build
```

Run application
```
npm run start
```

Run tests
```
npm run test
```
